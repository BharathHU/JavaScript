# JavaScript Learning Repository

## Overview
A topic-wise collection of JavaScript examples (mostly browser-based). The repo includes:
- **Pure JavaScript files (`.js`)** demonstrating core concepts
- **Browser demos (`.html`)** with embedded JavaScript for DOM, events, forms, async/await, etc.

Perfect for learning by opening files and experimenting.

## Quick Start
### Option A: Run an HTML demo (recommended)
1. Open any file ending with **`.html`** in your browser.
2. The examples are self-contained (no build tools needed).

Examples to try:
- `JS DOM Methods/DomQuerySelector.html`
- `javaScriptEvents/Onclick.html`
- `Form Validation/Form Validation/Example.html`

### Option B: Explore JavaScript files
- Open any **`.js`** file in VSCode to read and learn.
- For DOM/event-related code, remember it must run **in a browser context** (usually via an `.html` page).

## Folder Index (what’s inside)
Your top-level topics are organized into folders like:
- `js Arrays/` (arrays, 2D arrays, array methods)
- `js datatypes/` (number/string/boolean/null/undefined/bigint/symbol)
- `js variables/` (var/let/const, scope)
- `js operators/` (arithmetic, comparison, logical, assignment)
- `js functions/` (function types, callbacks, higher-order, IIFE, etc.)
- `js objects/` and `Objects/Objects/` (object patterns, CRUD, Date, math)
- `FlowControlStatements/` (if/else, switch)
- `Loops/` (loop examples)
- `strings/` (string methods)
- `JSON/` (JSON parsing/stringifying)
- `Template Literals/` and `Multi-Line Strings/`
- `Destructing of Array or Object/`
- `Spread Operator/` and `RestParameter/`
- `copyMetods/` (shallow vs deep copy)
- `thiskeyword/`
- `try catch block finally/` (error handling)
- `JS DOM Methods/` (getElementBy*, querySelector, create/remove nodes, attributes)
- `javaScriptEvents/` (event handler examples + `Event` object)
- `add event through javaScript/` (event listeners)
- `EventDelegation/` (delegation)
- `Event bubling/` (bubbling + capturing)
- `Asynchronous programming/`, `Promise in javascript/` (async/await, fetch, promises)
- `Synchronous programming/`
- `Timer/` (setTimeout/setInterval)
- `Form Validation/` (form validation demos)
- `Output/` (alert/console examples)

## How to Navigate
Use VSCode Explorer to jump to a topic folder, then open a related `.html` file (for browser behavior) or `.js` file (for logic examples).

## Getting Started (Terminal)
If you want to open the first example quickly:
```bash
start "js Arrays/array-literals.js"
```

Happy coding! 🚀
